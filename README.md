# SpaceX - 交互式演示系统

## 项目简介
这是一个基于 Vue 2 + Element UI 的审计作业系统交互演示，包含完整的项目管理、客户管理、合同管理、作业中心等功能模块。

## 功能模块

### 1. 项目管理
- 项目列表与详情
- 38种业务流程状态
- 项目进度跟踪

### 2. 客户管理
- 客户信息维护
- 客户状态跟踪

### 3. 合同管理
- 合同列表与详情
- 合同状态管理

### 4. 作业中心
- **数据准备**
  - 数据采集（财务数据、发票数据、资金数据）
  - 数据初始化（标准科目对应、底稿数据源设置、关联方设置、数据处理设置）
  - 数据校验（科目/凭证/账簿/试算平衡/报表校验）
  - 账套信息
- **风险评估**
- **作业工具**
- **底稿管理**

### 5. 设置中心
- 业务类型配置
- 报告文号规则
- 注会信息
- 通知设置
